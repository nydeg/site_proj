Создание MP-3 плеера.
MP-3 плеер должен включать в себя:
1. Окно авторизации с возможностью входа и регистрации. При неудачной авторизации должно возникнуть окно,
в котором говорится об ошибке;
2. При успешной регистрации/входе должно появиться окно "Меню", в котором можно добавлять файлы для проигрывания,
запускать случайный файл,
запускать выбранный файл по нажатию и выходить из аккаунта;
3. При запуске файла должно появляться окно с воспроизведением музыки из выбранного файла. В этом окне должны быть
реализованы следующие возможности: перемотка,
следующее аудио, предыдущее аудио, пауза, повтор воспроизведения, случайное воспроизведение (по умолчанию
воспроизведение по порядку).
4. Размеры окон фиксированные;
5. Расстановка кнопок, их размеры и прочее произвольно;
6. Код должен быть осуществлен на языке программирования Python с использованим библиотеки PyQt5.
Другие библиотеки не запрещены.
